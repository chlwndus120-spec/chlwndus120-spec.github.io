무중단 스마트 복합 주차관제

Seamless Unified Smart Complex Parking Web Service

AI 기반 번호판 인식, 통합 결제, 방문 예약, 상가 할인권을 하나의 플랫폼으로 묶은 주상복합 환경 맞춤 스마트 주차 관제 시스템. Toss Payments 기반 키오스크 사전정산 E2E 흐름을 구현하고, Redis 분산 락과 latestOrderId 검증으로 중복 결제 및 데이터 정합성 문제를 해결했습니다.

// 담당 역할

  • 키오스크 사전정산 흐름 구현 (차량 검색 → 요금 계산 → Toss 결제 / 포인트 분기 → 정산 확정)
  • Redis 기반 중복 결제 방지 락 (SETNX, TTL fallback, finally 해제 보장)
  • latestOrderId 검증으로 이전 탭/결제창의 승인 차단 및 금액 변조 방지
  • 조건부 UPDATE로 포인트 중복 차감 방지 및 음수 잔액 원자적 처리
  • 요금정책 버전 관리 · 매분 스케줄러 자동 활성화 구현
  • 정산 통계 (순매출 · 결제수단별 분석 · 할인권 가치 분리) 구현
  • 외부 PG 승인과 내부 DB 트랜잭션 분리로 커넥션 점유 최소화
  • Kiosk 챗봇 화면별 보조 기능 및 React/Zustand 상태 관리
